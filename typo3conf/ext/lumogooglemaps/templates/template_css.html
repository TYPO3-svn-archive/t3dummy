CSS-based template for plugin "tx_lumogooglemaps_pi1"

Template for positioning of map and sidebar

Available marker

The following markers are available:
###MAP_HEADERJS###
Marker is replaced with header JavaScript
###MAP_MAPJS###
Marker is replaced with JavaScript for map initialization
###MAP_MAP###
Marker is replaced by HTML and JavaScript for the map itself
###MAP_SIDEBAR###
Marker is replaced by HTML for the sidebar

Templates for sidebar

Available markers

The following markers are available:
###SIDEBAR_LIST###
Markers wrap the whole list of sidebar items
###SIDEBAR_ITEM###
Markers wrap a single sidebar item
###SIDEBAR_ITEM_LINK###
Marker is replaced with JavaScript function call to show item in map; remember to prefix by "javascript:" when used in a HTML link tag's href attribute
###SIDEBAR_ITEM_TITLE###
Marker is replaced with item's title (name field from corresponding tt_address item)
